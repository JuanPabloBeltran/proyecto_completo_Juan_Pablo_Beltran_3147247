# proyecto_completo_Juan_Pablo_Beltran_3147247

Proyecto Final – Sistema Web

Despliegues

Frontend: Netlify

Backend: Render

Base de datos: Railway (PostgreSQL)

Tecnologías

Frontend: HTML, CSS, JavaScript (o React, si aplica)

Backend: Node.js / Express (o Python Flask, si aplica)

Base de datos: PostgreSQL

Control de versiones: Git y GitHub

Instalación en local

Clonar el repositorio:

git clone https://github.com/JuanPabloBeltran/proyecto_completo_Juan_Pablo_Beltran_3147247.git

Backend
cd backend
npm install      (o pip install -r requirements.txt)
npm start        (o python app.py)

Frontend
cd frontend
npm install
npm run dev

Variables de entorno

Crear un archivo .env en el backend con la cadena de conexión de Railway:

DATABASE_URL=postgresql://usuario:contraseña@host:puerto/base

Estructura del repositorio
/frontend
/backend
